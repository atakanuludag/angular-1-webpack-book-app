# Book App

### Components
  - AngularJS 1.7.x
  - Webpack 4
  - SASS
  - Bootstrap

### Installation
```sh
cd angular-1-webpack-book-app
npm install --save
npm install --save-development
```

### Run the Project
```sh
cd angular-1-webpack-book-app
npm start
```

### Production the Project
```sh
cd angular-1-webpack-book-app
npm run build
```
